# Telecom Customer Churn & Revenue Impact Analysis

## Overview
This project analyzes telecom customer data to understand churn behavior and quantify its financial impact. 
The objective was to identify high-risk customer segments and measure revenue loss caused by churn.

## Dataset
- 7,000+ customer records
- Features include contract type, tenure, internet service, monthly charges, and total charges

## Tools Used
- SQL (PostgreSQL)
- Python (Pandas, NumPy, Matplotlib)
- Power BI (DAX, Data Modeling)

## Project Workflow
1. Data cleaning and preprocessing in Python
2. Exploratory data analysis (EDA)
3. SQL queries to calculate churn metrics and segment analysis
4. Revenue impact calculation
5. Dashboard creation in Power BI for KPI monitoring

## Key Findings
- Overall churn rate: 26.54%
- Fiber optic + Month-to-month customers show highest churn (~42%)
- Total churn revenue loss: ₹2.86M
- High-risk segment contributes ~60% of churn revenue

## Business Insight
Contract type and internet service significantly influence churn behavior. 
Targeted retention strategies for high-risk customers can help reduce revenue loss.
